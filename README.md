# Windows Timer Widget

가벼운 데스크탑용 카운트다운 타이머 위젯입니다. Python 기본 GUI(Tkinter)만 사용해서 별도 패키지 설치가 필요 없습니다.

## 실행

1. `run_timer_widget.bat`을 더블클릭합니다.
2. 또는 터미널에서 아래 명령을 실행합니다.

```powershell
python .\timer_widget.py
```

## 사용법

- 타이머 숫자나 상단 바를 드래그해서 위치를 옮길 수 있습니다.
- 듀얼 모니터에서도 보조 모니터 쪽으로 드래그 이동할 수 있습니다.
- `assets/notoow_logo.png`가 제목줄 로고와 앱 아이콘으로 표시됩니다.
- 창 제목은 남은 시간과 현재 상태를 함께 표시합니다.
- 실행, 마지막 10초, 완료, 일시정지 상태에 따라 시간 색과 진행바 색이 바뀝니다.
- 실행 중 앱이 재시작되면 저장된 종료 시각 기준으로 남은 시간을 이어갑니다.
- 이미 실행 중일 때 `run_timer_widget.bat`을 다시 열면 기존 창을 찾아 화면 안쪽으로 복구하고 앞으로 가져옵니다.
- 테두리 없는 위젯이지만 Windows 작업 표시줄과 Alt+Tab에서 찾기 쉽도록 앱 창 스타일을 적용합니다.
- `5m`, `10m`, `25m`, `50m`, `60m` 버튼은 한 줄에 가로로 배치됩니다.
- 각 시간 버튼은 반으로 나뉩니다. 왼쪽 반은 빼기, 오른쪽 반은 더하기입니다.
- 각 시간 버튼의 좌우 끝에 있는 `<`, `>` 표시는 빼기/더하기 방향만 보여주며, 클릭 영역은 정확히 반반입니다.
- 하단 입력 영역은 `min` / `sec` 일반 입력칸으로 구분되어 있으며, 위아래 스핀 화살표는 없습니다.
- 입력칸을 클릭하면 기존 숫자가 선택되어 바로 새 숫자로 덮어쓸 수 있습니다.
- 숫자 입력은 엔터를 누르지 않아도 즉시 타이머에 반영됩니다. `sec`는 0~59초만 받습니다.
- `시작`, `일시정지`, `정지`, `리셋` 버튼으로 타이머를 제어합니다. `정지`는 표시를 `00:00`으로 바꾸고, `리셋`은 입력된 시간으로 되돌립니다.
- 현재 상태에서 의미 없는 제어 버튼은 자동으로 비활성화됩니다.
- 완료 팝업에서는 `5분 더`, `다시 시작`, `닫기`를 바로 선택할 수 있습니다.
- `Mini`로 시간 추가 영역만 접을 수 있고, 접힌 상태에서는 버튼이 `Full`로 바뀝니다.
- `Pin` / `Top`으로 항상 위 옵션을 바꿉니다.
- 우상단 `⚙`에서 빠른 버튼 값, 완료 소리, 사용자 WAV, 반복 횟수, 투명도, 위치 초기화, Windows 시작 시 실행을 설정합니다.
- 우클릭 메뉴에서 투명도, 항상 위, 종료를 조정할 수 있습니다.
- 단축키: `Space` 시작/일시정지, `S` 정지, `R` 리셋, `Esc` 종료.
- 숫자 입력칸, 설정창, 완료 팝업에 포커스가 있을 때는 메인 타이머 단축키가 실행되지 않습니다.

위치, 투명도, 시간 설정은 변경 후 자동 저장되며 `timer_widget_state.json`에 기록됩니다. 저장은 임시 파일을 거쳐 반영되고, 설정 파일이 깨져 있으면 `timer_widget_state.invalid-*.json`으로 보관한 뒤 기본값으로 복구합니다.
